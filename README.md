# Linux Lab 🐧

> Plataforma de educación técnica para Linux Essentials, LPIC-1 y certificaciones.

Aprende Linux **haciendo**, no mirando. Laboratorios prácticos, feedback instantáneo y un agente inteligente que te guía en cada paso.

## 🎯 La Propuesta

Linux Lab no es otro curso de videos. Es una plataforma donde:

- **Practicas comandos reales** en sandbox seguro
- **Recibes feedback instantáneo** sobre lo que haces
- **Sigue un agente tutor** que detecta errores y sugiere el siguiente paso
- **Sigues rutas certificadas** Linux Essentials → LPIC-1 → Cloud Basics
- **Ves tu progreso** en tiempo real

## 🚀 Empezar Rápido

```bash
git clone https://github.com/Losd3/Web-Linux-Proyect.git
cd Web-Linux-Proyect
python -m http.server 8000
```

Luego abre http://localhost:8000
