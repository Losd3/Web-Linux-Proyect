1.3 Linux is Open Source

Entender el valor de Linux implica reconocer por qué el acceso al código fuente es una pieza clave de su éxito.

Progreso del módulo: 3 de 3 — has completado el módulo.

Históricamente, la mayor parte del software se ha distribuido bajo licencias de código cerrado. Esto significa que los usuarios tienen permiso para utilizar el programa en forma de código máquina, pero no pueden ver ni acceder al código fuente. En muchos casos, la licencia incluso prohíbe de manera explícita intentar realizar ingeniería inversa para reconstruir el código fuente y comprender cómo funciona el software.

El desarrollo de Linux está estrechamente ligado al surgimiento del software de código abierto. Este enfoque pone al código fuente en el centro del proceso. La filosofía del código abierto defiende que los usuarios tienen el derecho de acceder al código fuente del software y modificarlo según sus propias necesidades.

Linux adoptó esta filosofía con un éxito notable. Linus Torvalds puso el código fuente del sistema —las instrucciones que permiten que la computadora funcione— a disposición de todos, lo que permitió que otras personas participaran activamente en el desarrollo de este sistema operativo emergente. Aunque no fue el primer proyecto creado por una comunidad de voluntarios, el hecho de que Linux se desarrollara desde cero permitió que sus primeros colaboradores influyeran directamente en su dirección. Los usuarios tomaban el código, realizaban mejoras y compartían esos cambios con la comunidad, lo que aceleró significativamente el desarrollo y ayudó a evitar errores cometidos en otros sistemas operativos.

Para reflexionar

El código fuente puede escribirse en cientos de lenguajes de programación distintos. Linux, en particular, está escrito en C, un lenguaje versátil y relativamente fácil de aprender, que además comparte sus orígenes con el UNIX original. Esta decisión, tomada mucho antes de que se demostrara su impacto, resultó ser clave para que Linux se adoptara de forma casi universal como el sistema operativo principal de los servidores de Internet.
